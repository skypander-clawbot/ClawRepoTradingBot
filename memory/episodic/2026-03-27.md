# 2026-03-27 - Email Sending Instructions

## How to Send Emails via Gmail (App Password) using Python smtplib or Himalaya

### 1. Using Python smtplib (reliable)
```python
import smtplib
from email.mime.text import MIMEText

msg = MIMEText("Body text")
msg['Subject'] = "Subject"
msg['From'] = "skypanderclawbot@gmail.com"
msg['To'] = "recipient@example.com"

server = smtplib.SMTP('smtp.gmail.com', 587)
server.starttls()
server.login('skypanderclawbot@gmail.com', '<APP_PASSWORD>')
server.send_message(msg)
server.quit()
```

- App Password stored securely: `~/.secrets/mail/gmail.pass` (chmod 600)
- Retrieve via: `cat ~/.secrets/mail/gmail.pass`

### 2. Using Himalaya CLI (if configured)
- Config: `~/.config/himalaya/config.toml` with IMAP/SMTP using `auth.cmd = "cat ~/.secrets/mail/gmail.pass"`
- Send: `cat << EOF | himalaya --account gmail template send\nTo: someone@example.com\nSubject: Test\nBody\nEOF`
- Or interactive: `himalaya --account gmail message write`

### Notes
- Enable 2FA on Gmail and generate an App Password (Mail).
- Store password in secret file, never in plain config.
- Test with small message before bulk sends.
